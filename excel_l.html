canvas : 버튼누르면 캔버스가 나옴, Bootstrap
버튼과 캔버스는 떨어져 있어도 상관없는데, header태그 안에 버튼이 있으면 안되는듯 (버그인듯)
버튼과 관련
data-bs-toggle="offcanvas" : 기능 사용을 위해 꼭 필요, 기능을 사용한다는 의미
data-bs-target="#offcanvasBottom" : 토글되는 캔버스의 ID
aria-controls="offcanvasBottom" : 캔버스가 나타나는 위치 (Bottom은 아래에서 위로 올라옴)
기본적으로 캔버스 이외 배경이 흐리게되고 흐린부분 스크롤이 안되게 되어 있다.
스크롤 되도록 : data-bs-scroll="true" == 기본값(설정없어도) false
배경클릭시 닫기 : data-bs-backdrop="true" == true(흐림+닫힘), false(흰색+안닫힘), static(흐리게 + 안닫힘)
