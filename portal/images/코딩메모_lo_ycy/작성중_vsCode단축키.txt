-----------------------------------------------------------------------------------
ctrl + f : 찾기
ctrl + / : 주석
shift + delete : 줄 삭제
shift + ctrl + L : 전체변경 (바꿀단어 범위선택후 누르면 전체적으로 커서가 깜빡이고
                   동시에 변경된다.
ctrl + 마우스좌클릭 : 클래스나 아이디 위에 컨트롤 누르면, css가 기록되어 있다면
                      해당 css 내용을 팝업시켜준다. (추가기능 깔아야 됨)
tab : 여러줄도 가능하며 전체 탭 오른쪽으로 이동(추가).
shift + tab : 여러줄도 가능하며 전체 탭 왼쪽으로 이동(삭제).
